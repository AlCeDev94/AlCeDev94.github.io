BlockRent

This project is a Real Estate NFT DApp that enables users to buy houses on the blockchain and manage them as NFTs stored in IPFS (InterPlanetary File System).

The project utilizes two IERC-721 smart contracts: one to manage minting and total supply, and the other one to handle all additional functions.

The process involves not only buyer and seller but also lender, inspector and appraiser to supervise the sale.
